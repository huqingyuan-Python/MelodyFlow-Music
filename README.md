## 功能特色

### 核心功能
- **本地音乐导入** - 支持导入本地音频文件（MP3、AAC、WAV 等）
- **在线音乐搜索** - 支持网易云音乐、QQ音乐在线搜索
- **Apple Music 风格界面** - 现代化设计，沉浸式体验
- **歌词显示** - 支持 LRC 格式歌词，自动滚动同步
- **日语罗马音** - 日语歌曲可选择显示罗马音对照
- **歌词翻译** - 支持显示歌词中文翻译
- **播放列表** - 创建、管理自定义播放列表
- **收藏功能** - 一键收藏喜欢的歌曲
- **播放速率控制** - 0.5x ~ 2.0x 变速播放
- **A-B 重复** - 设置循环区间，适合学习语言
- **睡眠定时器** - 15/30/45/60 分钟自动停止
- **迷你播放器** - 悬浮小窗口，不遮挡界面
- **播放统计** - 记录每首歌播放次数
- **键盘快捷键** - 空格播放/暂停，方向键控制
- **歌曲下载** - 一键下载当前播放歌曲

### 在线音乐
- **网易云音乐** - 搜索并播放海量网易云音乐
- **QQ音乐** - 支持QQ音乐库
- **VIP 曲目直接听** - 会员歌曲无需会员，一键畅听
- **自动封面** - 自动获取歌曲封面
- **自动歌词** - 自动获取歌曲歌词

### 界面特色
- **四套主题** - 深色、浅色、粉色、蓝色
- **多语言** - 中文、English、日本语
- **响应式设计** - 完美适配桌面和移动设备
- **流畅动画** - 平滑过渡，动效细腻
- **全屏播放器** - Apple Music 风格全屏歌词视图

### 数据管理
- **本地存储** - 所有数据保存在浏览器本地
- **数据导出** - 一键备份音乐库和设置
- **数据导入** - 从备份文件恢复数据

## 快速开始

三种方式，根据需求任选其一。

### 方式一：在线搜索版（推荐，无需安装任何软件）

1. 直接用浏览器打开 `index.html`
2. 选择平台（网易云/QQ音乐）
3. 输入关键词搜索
4. 点击播放按钮立即播放

> 会员歌曲无需会员，直接播放

---

### 方式二：局域网部署版（一键启动）

适合希望局域网内多设备（电脑+手机）都能使用的用户。

| 平台 | 脚本文件 | 运行方式 |
|------|---------|---------|
| Windows | `启动服务.bat` | **双击运行** |
| macOS | `start-server-macos.sh` | 终端执行 `chmod +x` 后运行 |
| Linux | `start-server-linux.sh` | 终端执行 `chmod +x` 后运行 |

脚本会自动：检测 Node.js → 安装依赖（国内镜像）→ 启动服务 → 显示 IP 地址

> 首次使用需在播放器设置中填写 API 地址：`http://127.0.0.1:3000`

局域网内其他设备访问：`http://<服务器IP>:3000`（API 地址也要对应填写）

---

### 方式三：纯本地版

1. 下载或克隆本项目
2. 用浏览器打开 `index.html` 即可使用
3. 手动添加本地音频文件

## 使用指南

### 在线搜索音乐
1. 点击顶部导航「在线搜索」
2. 选择平台（网易云/QQ音乐）
3. 输入关键词搜索
4. 点击播放按钮立即播放，或 + 添加到资料库

### 导入本地音乐
1. 点击右上角「添加音乐」按钮
2. 选择本地音频文件
3. 音乐将自动添加到资料库

### 添加歌词
1. 播放音乐后，点击歌词区域的「添加歌词」按钮
2. 输入 LRC 格式的歌词
3. 格式示例：
```
[00:12.34]第一句歌词
[00:15.67]第二句歌词
[00:18.90]第三句歌词
```

### 日语罗马音和翻译
- 当检测到日语歌曲时，会自动显示罗马音开关
- 可开启「显示翻译」查看歌词翻译
- 可在设置中开启「默认显示罗马音」

### 创建播放列表
1. 点击侧边栏「+」或进入「播放列表」页面
2. 输入播放列表名称
3. 从歌曲列表添加歌曲到播放列表

### 键盘快捷键
| 按键 | 功能 |
|------|------|
| `Space` | 播放/暂停 |
| `←` / `→` | 上一曲/下一曲 |
| `Shift + ←` / `→` | 快退/快进 10 秒 |
| `↑` / `↓` | 音量增加/减少 |
| `M` | 静音切换 |
| `L` | 循环模式切换 |
| `F` | 全屏播放器切换 |

### A-B 重复（学习神器）
1. 在全屏播放器中点击「重复」按钮设置 A 点
2. 再次点击设置 B 点
3. 歌曲会在 A-B 区间内循环播放
4. 点击「清除」结束循环

### 睡眠定时器
1. 点击底部栏的月亮图标
2. 选择定时时长（15/30/45/60 分钟）
3. 定时开启后会显示剩余时间
4. 到时自动停止播放
